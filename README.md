# Tesis-MSDS
Repositorio scripts en Python y R de mi tesis de MSc. in Data Science

Revise detalles del commit para descripción de cada carpeta/archivo

R_tesis es el código en R para toda la implementación de ARDL/ECM junto con sus respectivas pruebas, estadísticos, etc.

P2Tesis es el código en Python que construye las métricas de la parte de optimización. Y su implementación se encuentra en el código MPL final.

Otros códigos son para la construcción de los datos.
